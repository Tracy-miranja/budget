# 📖 Blog Application <a name="about-project"></a>


**Budget Application** The Ruby on Rails capstone project (remember what they are?) is about building a mobile web application where you can manage your budget: you have a list of transactions associated with a category, so that you can see how much money you spent and on what.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:


```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd Documents
  git clone https://github.com/Tracy-miranja/budget.git
```
### Install

Install this project with:


```sh
  cd budget-Application
  gem install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

Example command:

```sh
  bin/rails test test/models/article_test.rb
```
